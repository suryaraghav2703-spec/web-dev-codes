Tutorial on loops

what is loops

Loops are a fundamental concept in programming that allow you to execute a block of code repeatedly, based on a condition. In JavaScript, there are several types of loops, including for loops, while loops, and do-while loops.

Examples of loops

Here are some examples of how loops work in JavaScript:

  • For Loop: A for loop is used when you know how many times you want to iterate.
  • While Loop: A while loop is used when you want to continue looping until a certain condition is met.
  • Do-While Loop: A do-while loop is similar to a while loop, but it guarantees that the code block will be executed at least once.

Types of loops in JavaScript

  • For Loop: Used for iterating a specific number of times.
  • Foreach Loop: Used for iterating over arrays or collections.
  • Map Loop: Used for applying a function to each element in an array and returning a new array.
  • Filter Loop: Used for creating a new array with all elements that pass the test implemented by the provided function.
  • Reduce Loop: Used for reducing an array to a single value by applying a function to each element.
  • While Loop: Continues until a condition is false.
  • Do-While Loop: Executes at least once before checking the condition.

Explaining each types

  • For Loop: A for loop is used when you know how many times you want to iterate. It consists of three parts: initialization, condition, and increment/decrement. For example: 
    for (let i = 0; i < 5; i++) {
    console.log(i);
}
  • Foreach Loop: The foreach loop is used for iterating over arrays or collections. It simplifies the process of looping through elements. For example: 
    const array = [1, 2, 3];
array.forEach((element) => {
    console.log(element);
});
  • Map Loop: The map loop is used for applying a function to each element in an array and returning a new array. For example: 
    const numbers = [1, 2, 3];
const doubled = numbers.map((num) => num * 2);
console.log(doubled); // [2, 4, 6]
  • Filter Loop: The filter loop is used for creating a new array with all elements that pass the test implemented by the provided function. For example: 
    const numbers = [1, 2, 3, 4, 5];
const evens = numbers.filter((num) => num % 2 === 0);
console.log(evens); // [2, 4]
  • Reduce Loop: The reduce loop is used for reducing an array to a single value by applying a function to each element. For example: 
    const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((acc, num) => acc + num, 0);
console.log(sum); // 15
  • While Loop: A while loop continues until a specified condition is false. For example: 
    let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
  • Do-While Loop: A do-while loop executes the code block at least once before checking the condition. For example: 
    let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
