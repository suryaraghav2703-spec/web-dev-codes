Arrays in JavaScript

Arrays are a fundamental part of JavaScript, allowing you to store and manipulate collections of data.

Creating Arrays

You can create an array using square brackets:

let fruits = ['apple', 'banana', 'orange'];

Accessing Array Elements

Access elements using their index:

console.log(fruits[0]); // apple

Array Methods

JavaScript provides many built-in methods for arrays:

  • push() - Adds an element to the end
  • pop() - Removes the last element
  • shift() - Removes the first element
  • unshift() - Adds an element to the beginning

Accessing Arrays

You can access array elements using their index:

console.log(fruits[0]); // apple

Finding the length

You can find the length of an array using the length property:

console.log(fruits.length); // 3

Changing the Values

You can change the value of an array element by accessing it via its index:

fruits[0] = 'kiwi';
console.log(fruits[0]); // kiwi

Array Methods in Depth

Let's explore some array methods in more detail:

toString()

Returns a string representing the array and its elements:

console.log(fruits.toString()); // apple,banana,orange

join()

Joins all elements of an array into a single string:

console.log(fruits.join(' - ')); // apple - banana - orange

pop()

Removes the last element from an array and returns that element:

let last = fruits.pop();
console.log(last); // orange
console.log(fruits); // ['apple', 'banana']

push()

Adds one or more elements to the end of an array and returns the new length of the array:

let newLength = fruits.push('mango');
console.log(newLength); // 4
console.log(fruits); // ['apple', 'banana', 'kiwi', 'mango']

shift()

Removes the first element from an array and returns that element:

let first = fruits.shift();
console.log(first); // apple
console.log(fruits); // ['banana', 'kiwi', 'mango']

unshift()

Adds one or more elements to the beginning of an array and returns the new length of the array:

let newLength = fruits.unshift('strawberry');
console.log(newLength); // 4
console.log(fruits); // ['strawberry', 'banana', 'kiwi', 'mango']

splice()

Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place:

fruits.splice(1, 2, 'blueberry', 'blackberry');
console.log(fruits); // ['strawberry', 'blueberry', 'blackberry', 'mango']

delete

Deletes an element from an array:

delete fruits[1];
console.log(fruits); // ['strawberry', undefined, 'blackberry', 'mango']

concat()

Used to merge two or more arrays:

let citrus = ['lemon', 'lime'];
let allFruits = fruits.concat(citrus);
console.log(allFruits); // ['strawberry', undefined, 'blackberry', 'mango', 'lemon', 'lime']

sort()

Sorts the elements of an array in place and returns the sorted array:

fruits.sort();
console.log(fruits); // ['blackberry', 'blueberry', 'kiwi', 'mango', 'strawberry']

slice()

Returns a shallow copy of a portion of an array into a new array object:

let citrus = fruits.slice(1, 3);
console.log(citrus); // ['blueberry', 'blackberry']

reverse()

Reverses the elements of an array in place:

fruits.reverse();
console.log(fruits); // ['mango', 'blackberry', 'blueberry', 'strawberry']

looping through Arrays

There are several ways to loop through arrays in JavaScript:

forEach()

Executes a provided function once for each array element:

fruits.forEach(function(item, index) {
   console.log(index + ': ' + item);
});

for...of

Iterates over iterable objects (including arrays):

for (let fruit of fruits) {
   console.log(fruit);
}

for...in

Iterates over the enumerable properties of an object:

for (let index in fruits) {
   console.log(index + ': ' + fruits[index]);
}

map()

Creates a new array populated with the results of calling a provided function on every element in the calling array:

let lengths = fruits.map(function(item) {
   return item.length;
});
console.log(lengths); // [10, 9, 10, 6]

filter()

Creates a new array with all elements that pass the test implemented by the provided function:

let longFruits = fruits.filter(function(item) {
   return item.length > 6;
});
console.log(longFruits); // ['blackberry', 'blueberry', 'strawberry']

reduce()

Executes a reducer function (that you provide) on each element of the array, resulting in a single output value:

let totalLength = fruits.reduce(function(accumulator, item) {
   return accumulator + item.length;
}, 0);
console.log(totalLength); // 35
