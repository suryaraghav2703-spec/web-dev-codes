JavaScript Variables

In JavaScript, variables are used to store data values. They are containers for holding data.

Declaring Variables

Variables can be declared using the var, let, or const keywords.

var, let, and const difference

  • var - function-scoped or globally-scoped, can be re-declared and updated
  • let - block-scoped, can be updated but not re-declared within the same scope
  • const - block-scoped, cannot be updated or re-declared

Variable Types

JavaScript variables can hold different types of values:

  • string - for text
  • number - for numbers
  • boolean - for true/false values

Example


    let name = "John";
    let age = 30;
    let isStudent = true;

Conclusion

Understanding variables is fundamental to programming in JavaScript. They allow you to store and manipulate data effectively.
