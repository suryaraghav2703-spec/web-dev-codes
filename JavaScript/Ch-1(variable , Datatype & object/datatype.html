why is type of null object?

In JavaScript, the type of null is considered an object due to a historical bug in the language. This means that when you check the type of null using the typeof operator, it returns "object". This can be confusing, but it's important to remember that null is a primitive value that represents the intentional absence of any object value.

objects

In JavaScript, an object is a standalone entity, with properties and type. It's similar to real-life objects, like a car, which has properties (color, brand, model) and behaviors (drive, stop, honk).

Objects can be created using object literals, the Object constructor, or classes (introduced in ES6).
