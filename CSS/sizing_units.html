  1. px (pixel) Screen ka chhota fixed dot. Responsive nahi hota. Use: borders, text, spacing jaha exact control chahiye.
  2. in (inch) Real duniya ke inches. (1in = 96px) Web me rarely use.
  3. pt (point) Print wali measurement. (1pt = 1/72 inch) Normal websites me use nahi hota.
  4. pc (pica) Print unit (1pc = 12pt). Web dev me almost kabhi nahi use hota.
  5. % (percent) Parent element ke size ka percentage. Use: Width/height jaha layout flexible chahiye.
  6. em Element ke font-size ke hisaab se size badalta hai. Compounding issue ho sakti hai (nested elements me size multiply hota rehta hai). Use: Padding/margin jaha font ke hisaab se spacing chahiye.
  7. rem Pure page ke root (html) font-size ke hisaab se. Predictable & best for responsive typography. Use: font-size, spacing, layout — sab rem me best lagta hai.
  8. vh Screen height ka percent. 1vh = screen height ka 1%. Use: Full-screen sections, hero banners.
  9. vw Screen width ka percent. 1vw = screen width ka 1%. Use: Responsive font-size, full-width layouts.
  10. vmin Screen ke chhote side (width/height) ka percent. Use: Perfect square / circular layouts.
  11. vmax Screen ke bade side ka percent. Use: Large responsive backgrounds ya elements.
